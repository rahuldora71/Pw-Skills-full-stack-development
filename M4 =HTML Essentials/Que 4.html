Ordered List of HTML Tags

  1. HTML: Hypertext Markup Language
  2. Head: Contains metadata about the HTML document
  3. Title: Specifies the title of the HTML document
  4. Body: Contains the visible content of the HTML document
  5. h1-h6: Defines headings of different levels
  6. p: Defines a paragraph of text
  7. a: Creates a hyperlink
  8. img: Embeds an image in the HTML document
  9. ul: Defines an unordered list
  10. ol: Defines an ordered list
