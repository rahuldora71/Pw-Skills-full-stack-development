1. HTML

    Primary Use Case Key Features/Benefits
    Building the structure of web pages
    • Simple and easy to learn
    • Compatible with all web browsers
    • Allows for semantic markup
    • Allows for inline styling

  2. CSS

    Primary Use Case Key Features/Benefits
    Styling and layout of web page
    • Allows for separation of content and presentation
    • Allows for responsive design
    • Allows for animations and transitions
    • Offers a wide range of styling options

  3. JS

    Primary Use Case Key Features/Benefits
    Adding interactivity to web pages
    • Allows for dynamic content
    • Allows for event handling
    • Allows for asynchronous communication
    • Allows for cross-browser compatibility
    • Allows for server-side scripting
